### LA PANTALLA. 
Varios tipos de pantalla que se pueden conectar a Arduino. 
- Daniel nosecuanto - hace cosas con pantallas.

#### oled. 
Pantalla que se bien, buena calidad. Es de 0,96 pulgadas, es de 128x64 (resolución). 
- SSD1306 (controlador), hace la pega de ocupar menos pines (Arduino), y se encarga de que se despliegen todos los pixeles que necesito.
  
Protocolos: 
- USB / Serial.
- i2C
(una forma de controlar la señal entre la pantalla al Arduino).

Que se necesita para hacer la conexión: i2C
- GND
- VCC
- SCL (pin5) Clock 
- SDA (pin4) Data 
( se pueden conectar un acelelometro o más cosas entre estos dos pines, ademas de la pantalla, por asi decirlo)
  
Tipos de pantallas: 
- LCD
- TFT

Librería: 
- Adafruit SSD1306.
```
#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>


// Definir tamaño del display
#define ANCHO_PANTALLA 128
#define ALTO_PANTALLA 64

// Dirección I2C típica (0x3C o 0x3D)
#define OLED_ADDR 0x3C  

// Crear objeto display
Adafruit_SSD1306 display(ANCHO_PANTALLA, ALTO_PANTALLA, &Wire);

int posicionVertical =0;

void setup() {
  // Iniciar la pantalla

   if(!display.begin(SSD1306_SWITCHCAPVCC, OLED_ADDR)) {
    Serial.begin(9600);
    Serial.println(F("No se encontró pantalla SSD1306"));
    for(;;); // se queda detenido
  }


}

void loop() {
posicionVertical++;

  display.clearDisplay();             // limpiar buffer
  display.setTextSize(1,5);             // tamaño del texto
  display.setTextColor(SSD1306_WHITE);// color del texto
  display.setCursor(ANCHO_PANTALLA/2, posicionVertical);          // posición (x,y)
  display.println("Hola mundo");      // escribir texto
  display.display();                  // mostrar en pantalla
 // nada que hacer en loop

if (posicionVertical > ALTO_PANTALLA){
  posicionVertical=0;
}

 delay(100);
}

```
